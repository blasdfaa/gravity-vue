# Spin

## Size

To control the size of the `Spin` use the `size` property. Default size is `m`.

## Properties

| Name         | Description                                               |              Type               |     Default     |
| :----------- | :-------------------------------------------------------- | :-----------------------------: | :-------------: |
| size         | Sets button size                                          |            `string`             |      `"m"`      |
| qa           | HTML `data-qa` attribute, used in tests                   |            `string`             |                 |
